# Unity ScriptableObject 教程 - 项目说明

## 📦 项目内容

本教程包含完整的 Unity ScriptableObject 学习资料，从基础到实战应用。

### 📄 文档文件

1. **ScriptableObject教程.md** - 主教程文档
   - 概念介绍
   - 基础到进阶的完整知识体系
   - 最佳实践和注意事项

2. **快速开始.md** - 5分钟快速上手指南
   - 快速入门步骤
   - 学习路线图
   - 项目结构说明

3. **常见问题.md** - 问题排查手册
   - 常见错误及解决方案
   - 调试技巧
   - 性能优化建议

### 💻 代码示例

#### 01_Basic - 基础示例
- `SimpleData.cs` - 最简单的 ScriptableObject
- `SimpleDataUser.cs` - 如何使用 ScriptableObject

#### 02_Intermediate - 进阶示例
- `GameConfig.cs` - 游戏配置系统
- `ItemData.cs` - 完整的物品系统

#### 03_Advanced - 高级示例
- `GameEvent.cs` - 事件系统核心
- `GameEventListener.cs` - 事件监听器
- `GameEventTrigger.cs` - 事件触发器

#### 04_Practical - 实战案例
- `EnemyData.cs` - 敌人数据配置
- `Enemy.cs` - 敌人控制器实现
- `SkillData.cs` - 技能数据系统
- `SkillSystem.cs` - 技能管理系统
- `LevelData.cs` - 关卡配置系统

### 📚 代码说明
- `Examples/代码说明.md` - 详细的代码使用说明和学习建议

---

## 🚀 如何使用

### 方式 1：阅读学习
1. 从 `快速开始.md` 开始
2. 阅读 `ScriptableObject教程.md` 了解完整概念
3. 查看 `Examples/代码说明.md` 理解代码结构
4. 遇到问题查阅 `常见问题.md`

### 方式 2：导入 Unity 项目
1. 将 `Examples` 文件夹复制到 Unity 项目的 `Assets` 目录
2. 等待 Unity 编译完成
3. 按照 `快速开始.md` 中的步骤创建第一个 ScriptableObject
4. 逐步学习每个示例
